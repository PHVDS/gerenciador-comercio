
O programa é iniciado através da classe Main.java que está programada para instanciar  a classe JLogin.java que é responsável de mostrar a tela de login e checar as credenciais do usuário fazendo consulta no banco de dados (senha esta criptografada), tenho uma classe chamada GeraCriptografia.java que criptografa e descriptografa dados de string. Quando o login é autorizado a janela principal do programa é carregada (JPrincipal.java) e junto com ela é carregada uma arvore (estrutura de dados) com todos os produtos cadastrados para ficar em memória e deixar o programa mais rápido e não tão dependente de varias consultas SQL.
A classe JPrincipal.java é a janela mais importante mas as outras tem papel fundamental no gerenciamento das coisas, para ajudar a navegação entra janelas eu uso um menu (classe Menu.java) tem algumas funções de chamadas de outras janelas, quando uma janela chama outra eu uso uma técnica de navegabilidade de classe, a maioria das vezes quando uma janela é invocada ela instancia e trata os dados nova janela, exibe e logo depois fecha a antiga, maior parte do tempo só uma janela estará aberta.
